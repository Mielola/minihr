Cara menggunakan laravel :
1. Instal terlebih dahulu vendor nya dengan "Composer Install". Jika ada error masukkan juga "npm install"
2. Masukkan database yang bernama "minihr.sql". Jika tidak ada database pada lampiran jalankan "php artisan migrate"
3. Sesuaikan koneksi database pada file .env
4. Jalankan project menggunakan "php artisan serve"
5. Berikut untuk akun yang bisa digunakan pada masing masing role
